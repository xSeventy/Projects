# Personal Budget

Реализуйте систему ведения личного бюджета.

Программа должна обрабатывать запросы двух типов:

- `date value` — учесть, что в указанную дату была заработана сумма `value`
- `from to` — вычислить прибыль за данный диапазон дат

Сначала во входных данных идут только запросы с информацией о заработках, а затем — только запросы на вычисление прибыли.

## Примечания

- Во всех диапазонах `from to` обе даты — `from` и `to` — включаются
- Все даты задаются в формате `YYYY-MM-DD`

## Формат ввода

В первой строке вводится количество дней с информацией о заработках `E`.

Далее следуют `E` запросов вида:

`date value`

где:
- `date` — дата
- `value` — сумма заработка за эту дату

После этого вводится количество запросов на вычисление прибыли `Q`.

Далее следуют `Q` запросов вида:

`from to`

где:
- `from` — начало диапазона
- `to` — конец диапазона

## Формат вывода

Для каждого запроса на вычисление прибыли выведите в отдельной строке одно целое число — прибыль за указанный диапазон дат.

## Ограничения

- Количество запросов `Q` — натуральное число, не превышающее `100000`
- Все даты корректны, с учётом високосных годов
- Все даты принадлежат интервалу от `1700` до `2099` года
- `value` — положительные целые числа, не превышающие `1000000`
- Время на обработку всех запросов: `4.9` секунды

## Подсказка

Используйте `std::partial_sum`, чтобы отвечать на каждый запрос, считая разность между двумя частичными суммами.
