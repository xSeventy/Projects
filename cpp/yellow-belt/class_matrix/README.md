# Matrix

В этой задаче нужно разработать класс `Matrix` для представления целочисленной матрицы.

## Что должен уметь класс

Класс `Matrix` должен поддерживать следующий интерфейс:

### Конструкторы
- **Конструктор по умолчанию**  
  Создаёт матрицу с нулём строк и нулём столбцов.

- **Конструктор от двух целочисленных параметров**  
  Принимает:
  - `num_rows` — количество строк
  - `num_cols` — количество столбцов

### Методы
- **`Reset(num_rows, num_cols)`**  
  Изменяет размер матрицы на заданный и обнуляет все её элементы.

- **`At(row, col)`**  
  Должен быть реализован в двух версиях:
  - константной — возвращает значение элемента;
  - неконстантной — возвращает ссылку на элемент, чтобы его можно было изменять.

  Нумерация строк и столбцов начинается с нуля.

- **`GetNumRows()`**  
  Возвращает количество строк.

- **`GetNumColumns()`**  
  Возвращает количество столбцов.

## Обработка ошибок

Класс должен выбрасывать стандартное исключение `out_of_range` в следующих случаях:

- если в конструктор или метод `Reset` передано отрицательное количество строк или столбцов;
- если метод `At` вызывается с индексами, выходящими за границы матрицы.

## Поддерживаемые операторы

Для класса `Matrix` должны быть определены следующие операторы:

### Ввод из потока `istream`
Формат ввода:
1. количество строк;
2. количество столбцов;
3. все элементы матрицы по строкам:
   - сначала элементы первой строки,
   - затем второй,
   - и так далее.

### Вывод в поток `ostream`
Матрица должна выводиться в том же формате, в котором читается:
- в первой строке — количество строк и столбцов;
- далее — элементы матрицы построчно.

### Оператор сравнения `==`
Возвращает `true`, если:
- размеры матриц совпадают;
- все соответствующие элементы равны.

Во всех остальных случаях должен возвращать `false`.

### Оператор сложения `+`
Принимает две матрицы и возвращает новую матрицу, равную их сумме.

Если размеры матриц различаются, оператор должен выбрасывать стандартное исключение `invalid_argument`.
