# Final Database

Финальная задача белого пояса по C++.

## Условие
Нужно реализовать простую базу данных, которая хранит пары:

- дата
- событие

Дата вводится в формате `Год-Месяц-День`.
Событие — непустая строка без пробелов внутри.

Программа должна читать команды из стандартного ввода и поддерживать операции:

- `Add date event` — добавить событие к дате
- `Del date event` — удалить одно событие
- `Del date` — удалить все события за дату
- `Find date` — вывести события за дату
- `Print` — вывести всю базу

Если на одну дату добавляют одинаковое событие несколько раз, хранить его нужно только один раз.
