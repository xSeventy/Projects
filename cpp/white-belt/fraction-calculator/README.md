# Fraction Calculator

Задача про калькулятор обыкновенных дробей.

## Условие
Нужно прочитать одну строку вида:

`дробь_1 операция дробь_2`

Дроби задаются в формате `X/Y`.
Операция — одна из `+`, `-`, `*`, `/`.

Программа должна:
- корректно выполнить операцию над дробями;
- вывести `Invalid argument`, если одна из дробей некорректна;
- вывести `Division by zero`, если происходит деление на ноль.
