# Daily Tasks

Задача про список дел по дням месяца.

## Условие
Нужно реализовать программу, которая хранит дела по дням текущего месяца и поддерживает команды:

- `ADD i s` — добавить дело `s` в день `i`
- `DUMP i` — вывести все дела дня `i`
- `NEXT` — перейти к следующему месяцу

Особенность команды `NEXT`:
- если в новом месяце дней больше, новые дни начинаются пустыми;
- если в новом месяце дней меньше, дела из "лишних" дней нужно перенести на последний день нового месяца.

Начальный месяц — январь.
Февраль всегда считается месяцем из 28 дней.
