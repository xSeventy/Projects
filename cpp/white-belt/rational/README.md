# Rational

Задача про класс `Rational`.

## Условие
Нужно реализовать класс для работы с рациональными числами.

Базовые требования:
- дробь должна храниться в сокращённом виде;
- знаменатель должен быть положительным;
- ноль должен храниться как `0/1`;
- должен быть конструктор по умолчанию и конструктор от числителя и знаменателя.

В расширенной версии задачи обычно добавляют:
- сравнение рациональных чисел;
- арифметические операции;
- ввод и вывод;
- обработку ошибок и исключения.

## Примечание
В исходном курсе задача разбита на несколько шагов.
В этой папке лежит одна итоговая реализация.
