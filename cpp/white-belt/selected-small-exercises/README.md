# Selected Small Exercises

Небольшой набор коротких задач из белого пояса.

## Состав
- `sort-by-module.cpp` — сортировка целых чисел по модулю
- `sort-without-register.cpp` — сортировка строк без учёта регистра
- `sorted-strings.cpp` — класс, который хранит строки и возвращает их в отсортированном порядке

## Краткие условия

### sort-by-module
На вход подаётся количество чисел и сами числа.
Нужно отсортировать массив по абсолютному значению элементов.

### sort-without-register
На вход подаётся количество строк и сами строки.
Нужно отсортировать их лексикографически, игнорируя регистр букв.

### sorted-strings
Нужно реализовать класс, который:
- добавляет строку в набор;
- возвращает все строки в отсортированном порядке.
